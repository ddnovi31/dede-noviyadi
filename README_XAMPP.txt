CARA PALING SIMPEL MENJALANKAN DI XAMPP:

1. Ambil File:
   - Saya sudah mengonfigurasi aplikasi ini untuk menghasilkan SATU FILE HTML saja.
   - Buka folder 'dist' dan ambil file 'index.html'.

2. Copy ke XAMPP:
   - Buka folder 'C:\xampp\htdocs'.
   - Copy file 'index.html' tersebut ke sana.
   - (Opsional) Rename menjadi 'kabel.html' agar tidak bentrok.

3. Jalankan:
   - Buka browser dan akses: http://localhost/index.html (atau http://localhost/kabel.html)

Selesai! Tidak perlu copy folder assets, karena semua CSS dan JS sudah digabung ke dalam satu file index.html tersebut.
